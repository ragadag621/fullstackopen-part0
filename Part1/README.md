


# Full Stack Open 2026 - Part 1 (React Fundamentals)

This repository contains my solutions for the first part of the Full Stack Open course. This stage was focused on mastering the core concepts of **React.js** and functional programming.

##  Exercises & Projects

### 1️⃣ Course Information (1.1 - 1.5)
- **Concept:** Breaking down a monolithic UI into reusable components.
- **Key Skills:** JSX, Props passing, and Component Architecture.

### 2️⃣ Unicafe (1.6 - 1.11)
- **Concept:** A feedback gathering system with real-time statistics calculation.
- **Key Skills:** `useState` hook, **Conditional Rendering**, and Event Handling.
- **Best Practices:** Implementing HTML tables for structured data presentation and ensuring UI clarity.

### 3️⃣ Anecdotes (1.12 - 1.14)
- **Concept:** A random programming anecdote generator with a dynamic voting system.
- **Key Skills:** Array manipulation (Immutability), `Math.random`, and `Math.max`.
- **Logic:** Finding the most voted item by tracking indices and updating state using the **Spread Operator** `[...]`.

##  Key Takeaways
* **Immutability:** Understanding that React state should never be mutated directly; always create a copy before updating.
* **Component-Based Logic:** Learning how to separate "Actions" (Buttons) from "Data Display" (Statistics) to write cleaner code.
* **Problem Solving:** Applying JavaScript logic (finding max values, generating random indices) within the React lifecycle.
